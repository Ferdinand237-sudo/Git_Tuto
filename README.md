# DOCUEMENTATION DU TUTO GITHUB AVEC GIT

# Initialiser un dépot
```bash
git init
git remote add origin SSH_DU_REPO
```
# Vérifier l'état d'un dépot
```bash
git status
```
# Rédiger un commit (les bonnes pratiques)

```
Titre du commit
Description de notre commit avec des informations surr l'évolution du projet
```

# Envoyer un commit sur un dépot distant
```bash
git add .
git commit -m "titre du commit"
git push origin main
```

## Création d'une branche
```bash
git checkout -b branch
```
